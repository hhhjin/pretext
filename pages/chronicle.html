SCENT OF AN IDEA:
Winding Contours

Some pages do not need CSS columns so much as a memory of where the text has already been. The fun begins when every line can steer around a shape without giving the browser the whole composition back.

A layout like this is less about imitation than leverage: one body of text, several live contours, and a surface that can be resized without collapsing into obedient boxes.

The first sketch smelled of cedar, toner, and hot dust from the projector. Someone dragged a contour across the page and the prose no longer wanted to live in rectangles.

Resize the surface and the same text finds a new route. Nothing here is measured back out of the browser after the fact; the page owns its own line breaks.

Built from repeated layoutNextLine() calls around fixed copy blocks and procedural artwork.
Manual userland reflow. The poster is one explicit layout pass: viewport reads, geometric routing, then absolute DOM writes.
Resize the window to recompose the field.
